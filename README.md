# NSC OT Calculator Mock

Web app แยกสำหรับคำนวณ OT และตรวจรายการที่พนักงานอยู่เกินเวลาเคสสุดท้าย

## วิธีเปิดดู

เปิดไฟล์:

`index.html`

ตอนนี้แยกเป็น 2 หน้า:

- `index.html` สำหรับ import Excel และคำนวณ OT
- `settings.html` สำหรับตั้งค่าเคสสุดท้ายของวันและกฎ OT รายแผนก
- `history.html` สำหรับดูประวัติข้อมูลจาก LINE LIFF และประวัติเคสสุดท้ายรายวัน

รูปแบบวันที่ที่แสดงในหน้าเว็บใช้ `dd/mm/yyyy` ทั้งหมด เช่น `08/06/2026`

## กฎที่ใส่ไว้

- แผนกผ่าตัด `OR`
  - ก่อน 23:00 ชั่วโมงละ 50 บาท
  - 23:00-08:00 ชั่วโมงละ 80 บาท
  - หลัง 08:00 ถือเป็นวันใหม่และขึ้น note
  - ออกหลังเคสสุดท้ายเสร็จได้ไม่เกิน 2 ชั่วโมง
- แผนกการตลาด `MKT`
  - ชั่วโมงละ 100 บาท
  - ออกหลังเคสสุดท้ายเสร็จได้ไม่เกิน 1 ชั่วโมง
- แผนกอื่น
  - ชั่วโมงละ 50 บาท
  - ออกหลังเคสสุดท้ายเสร็จได้ไม่เกิน 1 ชั่วโมง

## Setting ที่เพิ่มแล้ว

- ตั้งค่าเคสสุดท้ายของวัน:
  - วันที่ของเคส
  - ทำอะไร
  - เสร็จกี่โมง
  - รายละเอียดเพิ่มเติม
- ค่านี้เก็บแยกตามวันที่ และจะใช้กับทุกคนที่ Check-in ผ่าน LINE ของวันนั้น ถ้า Excel ไม่ได้ใส่ `Last_Case_End` รายคน
- ตั้งค่ากฎ OT รายแผนก:
  - รหัสแผนก
  - ค่า OT ต่อชั่วโมง
  - อยู่หลังเคสสุดท้ายได้กี่ชั่วโมง
  - เปิด rate พิเศษ 23:00-08:00 ได้
  - เพิ่มแผนกใหม่ได้ เช่น `HR`, `CS`, `SALE`
- ตั้งค่ากฎมาสาย:
  - หักนาทีละกี่บาท ค่าเริ่มต้น 2 บาทตาม requirement

ค่าที่แก้ในหน้า Setting ถูกเก็บจำลองใน `localStorage`

## History

- History จาก LIFF อ่านจาก key `nsc-liff-attendance-history`
- LIFF mock จะเพิ่ม history ทุกครั้งที่พนักงานบันทึกเข้างาน/ลา/หยุด และอัปเดตเมื่อกดตอบ OT
- History เคสสุดท้ายอ่านจาก setting รายวันที่บันทึกใน `settings.html`

## รูปแบบ Excel ที่แนะนำ

รองรับ 2 แบบ:

1. ไฟล์สแกนนิ้วรายเดือนแบบที่ row 2 เป็นวันที่ เช่น `4-1`, `4-2`, `4-30` และคอลัมน์ A เป็น `ชื่อบุคคล`
2. ไฟล์แบบมีคอลัมน์ดังนี้

| Name | Department | Start_Time | Scan_Out | Last_Case_End |
|---|---|---|---|---|
| Grace ABC | MKT | 09:00 | 20:30 | 19:00 |

ถ้าใช้ไฟล์สแกนนิ้วรายเดือน ระบบจะใช้วันที่จากหัวตาราง Excel และไปจับคู่กับ History จาก LIFF ด้วยชื่อพนักงาน + วันที่ เพื่อดึงเวลาเริ่มงานและแผนก

ถ้าไม่มี `Last_Case_End` รายแถว ระบบจะใช้ค่าเวลาเคสสุดท้ายเสร็จจากช่องด้านบน

## หมายเหตุ

การอ่าน Excel ใช้ SheetJS CDN ใน browser ถ้าเปิดแบบไม่มีอินเทอร์เน็ตอาจอ่านไฟล์ Excel ไม่ได้ แต่ยังใช้ปุ่มโหลดข้อมูลตัวอย่างได้

## การใช้เคสสุดท้ายกับ OT

ระบบจะใช้เวลาเคสสุดท้ายรายวัน + buffer ของแผนกเป็น cutoff สำหรับ OT ที่คิดเงินได้

ตัวอย่าง: เข้างาน 08:00 เลิกปกติ 17:00 เคสสุดท้ายเสร็จ 21:00 แผนกที่ buffer 1 ชั่วโมงจะคิด OT ได้ถึง 22:00 ถ้าสแกนออก 00:00 ระบบจะแสดง warning ส่วนที่เกิน 22:00 และไม่คิดเงิน OT อัตโนมัติจนกว่า HR ตรวจ
